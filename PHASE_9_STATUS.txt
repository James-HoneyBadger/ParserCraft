Mobile Platform Usage:
```python
from hb_lcs.mobile_cloud_analytics import MobilePlatform, MobilePlatformManager

manager = MobilePlatformManager()
config = manager.create_mobile_config(
    platform=MobilePlatform.IOS,
    app_name="MyApp",
    bundle_id="com.example.myapp",
)
build = manager.package_app(config)
print(f"Build artifacts: {build['artifacts']}")
```

Cloud Deployment Usage:
```python
from hb_lcs.mobile_cloud_analytics import CloudProvider, CloudDeploymentManager

manager = CloudDeploymentManager()
config = manager.create_deployment_config(
    provider=CloudProvider.AWS,
    region="us-east-1",
    instance_type="t3.medium",
)
deployment = manager.deploy(config, app_name="my-app", version="1.0.0")
print(f"Endpoints: {deployment['endpoints']}")
```

Analytics Usage:
```python
from hb_lcs.mobile_cloud_analytics import AnalyticsTracker, MetricType

tracker = AnalyticsTracker()
session_id = tracker.start_session(user_id="user123")

tracker.track_event("page_view", user_id="user123", session_id=session_id,
                   properties={"page": "/home"})

tracker.record_metric("response_time", MetricType.TIMER, 125.5,
                     tags={"endpoint": "/api/users"})

report = tracker.get_analytics_report()
print(f"Total events: {report['total_events']}")
```
